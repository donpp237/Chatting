# Chatting

## Purpose

TCP/IP 소켓 통신 학습을 위한 채팅 서버 및 클라이언트 생성

## Content

### Server
접속 사용자들에게 데이터 송신 및 수신하는 중간 역할을 담당하며,
접속 사용자의 대화명을 확인할 수 있도록 구성.

### Client
서버로의 데이터 송신 및 수신 역할을 담당.
